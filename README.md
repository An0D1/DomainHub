# Асеев А.А. практическая работа №8

## Задание 1. Главная страница сайта

1) **Название файла:** `index.html`

2) **Путь:** корневая папка проекта

   **Что делает программа:**  
   Главная страница сайта по продаже доменных имен «DomainHub». Содержит навигационное меню, поиск домена, карточки популярных доменов, преимущества компании и форму обратной связи.

   **Какой метод иллюстрирует:**  
   Структуру HTML-документа, подключение стилей и скриптов, использование семантических тегов (`header`, `main`, `section`, `footer`), создание адаптивной верстки через CSS.

---

## Задание 2. JavaScript-файл (форма обратной связи)

1) **Название файла:** `form.js`

2) **Путь:** `assets/js/form.js`

   **Что делает программа:**  
   Обрабатывает отправку формы обратной связи. При нажатии на кнопку «Отправить сообщение» собирает данные из полей (имя, email, сообщение), формирует письмо и открывает Mail.ru с предзаполненными полями. В случае ошибки выводит сообщение в консоль.

   **Какой метод иллюстрирует:**  
   Документирование JavaScript-кода с использованием JSDoc:
   - `@file` — описание файла
   - `@author` — имя разработчика
   - `@version` — версия скрипта
   - `@typedef` и `@property` — описание структуры данных
   - `@param` — описание параметров функции
   - `@this` — контекст выполнения
   - `@returns` — возвращаемое значение
   - `@see` — ссылки на связанные ресурсы

## Задание 3. Основные стили сайта

1) **Название файла:** `style.css`

2) **Путь:** `assets/css/style.css`

   **Что делает программа:**  
   Описывает внешний вид всех элементов сайта: цвета, шрифты, отступы, размеры, эффекты при наведении. Использует CSS-переменные для удобства поддержки.

   **Какой метод иллюстрирует:**  
   Организацию CSS-кода с комментариями, разделение на логические блоки, использование переменных (`:root`), стилизацию скроллбара, создание сеток через flexbox.

---

## Задание 4. Стили для адаптивности

1) **Название файла:** `media.css`

2) **Путь:** `assets/css/media.css`

   **Что делает программа:**  
   Обеспечивает корректное отображение сайта на разных устройствах:
   - При ширине экрана до 1280px — уменьшение размеров, изменение расположения элементов
   - При ширине до 768px — скрытие меню, вертикальное выравнивание карточек
   - Поддержка `prefers-reduced-motion` для пользователей, чувствительных к анимации

   **Какой метод иллюстрирует:**  
   Использование медиа-запросов (`@media`) для создания адаптивного дизайна.

---

## Задание 5. Анимации

1) **Название файла:** `animations.css`

2) **Путь:** `assets/css/animations.css`

   **Что делает программа:**  
   Добавляет визуальные эффекты:
   - `pulse` — пульсация кнопки поиска и карточек спецпредложений
   - `float` — плавное покачивание карточек
   - `slideIn` — появление элементов с задержкой
   - `rotate` — вращение иконок в секции «Преимущества»

   **Какой метод иллюстрирует:**  
   Создание CSS-анимаций через `@keyframes`, управление задержками анимации (`animation-delay`), остановка анимации при наведении.
